KSeF2 CLI jest interfejsem wiersza polecen dla SDK Pythona `ksef2`.
Pozwala uruchamiac najwazniejsze przeplywy KSeF z terminala: uwierzytelnianie,
zapytania o faktury, wysylke online i wsadowa, tokeny, uprawnienia, limity oraz
dane TEST.

Pelna dokumentacja CLI jest obecnie utrzymywana po angielsku:

- [Installation](getting-started/installation.md)
- [Quickstart](getting-started/quickstart.md)
- [Authentication](guides/authentication.md)
- [Configuration](guides/configuration.md)
- [Invoice workflows](guides/invoices.md)
- [Command reference](reference/commands.md)

Stabilna dokumentacja przeplywow SDK znajduje sie w [dokumentacji SDK](../sdk/intro.md).
